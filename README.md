# imessage-undeleter

### Supported Features

This crate supports every iMessage feature as of macOS 15.5 (24F74) and iOS 18.5 (22F75):

- iMessage, RCS, SMS, and MMS
- Multi-part messages
- Replies/Threads
- Formatted text
- Attachments
- Expressives
- Tapbacks
- Stickers
- Apple Pay
- Group chats
- Digital Touch
- URL Previews
- Audio messages
- App Integrations
- Edited messages
- Handwritten messages
